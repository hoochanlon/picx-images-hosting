# ④ 第四步：开始使用

配置完成后，你就可以开始使用图床服务了！

## 功能说明：

- **快速上传：** 点击导航栏的上传按钮，选择图片文件上传
- **上传管理：** 管理已上传的文件，支持删除、重命名、复制链接
- **图片浏览：** 在首页浏览所有图片，支持搜索和筛选
- **CDN 加速：** 使用 jsDelivr CDN 加速图片加载

## 链接格式：

```
GitHub Pages 链接：
https://你的用户名.github.io/仓库名/图片路径

CDN 链接（推荐）：
https://cdn.jsdelivr.net/gh/你的用户名/仓库名@分支名/图片路径
```

## 配置自定义域名（可选）：

如果你有自己的域名，可以在 `config.js` 中配置：

```javascript
CUSTOM_DOMAINS: [
  'your-domain.com'
]
```

> **完成！** 现在你已经成功部署了图床服务，可以开始上传和管理图片了。
