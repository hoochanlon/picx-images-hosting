# ① 第一步：创建一个 GitHub 仓库

在 GitHub 上创建一个新仓库用于存储图片。

## 操作步骤：

1. 登录 GitHub，点击右上角 **"+"** → **"New repository"**
2. 填写仓库名称（如：`picx-images-hosting`）
3. 选择可见性：**Public**（公开）或 **Private**（私有）
4. 点击 **"Create repository"** 完成创建

:::tip
仓库名称建议使用小写字母和连字符，避免使用特殊字符。
:::

## 创建 Release（推荐）

创建 Release 后可以使用更稳定的 CDN 链接：

1. 进入仓库，点击右侧 **"Releases"**
2. 点击 **"Create a new release"**
3. 输入版本号（如：`v1.0.0`）
4. 点击 **"Publish release"**

:::important
记下你的仓库地址，格式为：
```
https://github.com/你的用户名/仓库名
```
后续配置会用到这个地址。
:::
